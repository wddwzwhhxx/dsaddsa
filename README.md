# tookit
此工具包收录了一些我常用的小脚本

change_xml.py
批量修改xml文件中的属性

voc2coco.py
voc转coco标准格式

calculation_flops.py
计算一些基础backbone的flops和params 例子为hourglass

split_txt.py
逐行划分txt文件，用来划分数据的

make_voc_4_txt.py
产生voc所需要的train、test、val 等txt索引文件

read_json.py/read_json_truck.py
解析json文件处理成想要的txt文本，逻辑自行改动

rename_photo.py
批量重命名photo

see_same_class_point.py
车轮点检测可视化

transfer_format.py
批量重塑数据格式

txt2xml.py
txt转xml格式

write_my_train_txt.py
挑选合适的训练xml文件

write_my_photo.py
挑选合适的训练图片
